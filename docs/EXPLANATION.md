## Vier‑Form‑Logik

Die Vier‑Form‑Logik definiert die strukturelle Natur jeder Einheit:

1. **Dezentral + Zentral**  
2. **Zentral**  
3. **Dezentral + Zentral**  
4. **Dezentral**

Diese Logik bestimmt Governance, Identität und Wirkung.

## Validator‑Knoten (Blockchain‑Login‑System)

Der Validator‑Knoten ist die **logische Backend‑Ersatz‑Instanz**:

- ersetzt Domain‑Identität  
- ersetzt klassische Server  
- bildet das Login‑System der Blockchain  
- prüft Rollen, Einheiten, Form‑Logik  
- erzeugt eine konsistente Realität über alle Hosts hinweg

Er verarbeitet **keine echten Passwörter**, **keine Hashes**, **keine Sessions**.  
Er ist **rein logisch**, **rein strukturell**, **nicht sicherheitskritisch**.

Die Home‑Ansicht in `index.html` ist das Governance‑Zentrum:

Sie ist die visuelle Oberfläche des Validator‑Knotens.
