## Validator‑Knoten (Blockchain‑Login‑System)

Der Validator‑Knoten ist die **zentrale logische Instanz** des Systems:

- ersetzt die Domain als Identitätsquelle  
- ersetzt klassische Backends  
- bildet das Login‑System der Blockchain  
- prüft Rollen, Einheiten, Form‑Logik, Governance  
- erzeugt eine konsistente Realität über alle Hosts hinweg

`index.html` ist nur Oberfläche.  
Der Validator ist die **Regel‑ und Identitätsinstanz**.
